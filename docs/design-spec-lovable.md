# Lovable 設計規格文件

> 根據 Lovable 設計稿截圖整理的 UI 規格

---

## 1. 設計系統 (Design Tokens)

### 色彩

| 用途 | 色碼 | 說明 |
|------|------|------|
| 頁面背景 | `#F5F0E8` | 暖奶油色 |
| 卡片背景 | `#FFFFFF` / `#FAF8F5` | 白色或近白 |
| 主要文字 | `#5D4E37` | 深棕色 |
| 次要文字 | `#9B8B7A` | 淺棕色 |
| 強調色 | `#A67C52` | 銅棕色 |
| 深強調 | `#6B5344` | 深棕 |
| 圖標背景 | `#F0EBE3` | 淺米色圓形 |
| 成功色 | `#5D8A66` | 綠色 (+數值) |
| 錯誤色 | `#C45C5C` | 紅色 (-數值/刪除) |
| 星級色 | `#D4A24C` | 金黃色 |

### 間距

| 名稱 | 值 | 用途 |
|------|-----|------|
| xs | 4px | 緊密間距 |
| sm | 8px | 小間距 |
| md | 12px | 標準間距 |
| lg | 16px | 區塊間距 |
| xl | 24px | 大區塊間距 |
| xxl | 32px | 頁面邊距 |

### 圓角

| 名稱 | 值 | 用途 |
|------|-----|------|
| sm | 8px | 小元素 |
| md | 12px | 按鈕、輸入框 |
| lg | 16px | 卡片、對話框 |
| xl | 20px | 大卡片 |
| full | 999px | 膠囊形狀 |

### 陰影

```css
/* 卡片陰影 */
shadow-card: 0 2px 8px rgba(0, 0, 0, 0.04);

/* 提升陰影 */
shadow-elevated: 0 4px 16px rgba(0, 0, 0, 0.08);
```

### 字體

| 用途 | 大小 | 粗細 |
|------|------|------|
| 頁面標題 | 22px | 700 |
| 區塊標題 | 18px | 600 |
| 卡片標題 | 16px | 600 |
| 內文 | 14px | 400 |
| 副標題 | 13px | 400 |
| 小標籤 | 12px | 500 |
| XP 獎勵 | 14px | 600 |

## 2. 元件庫

### 2.1 圖標圓形 (IconCircle)

```
┌─────────┐
│  (icon) │  40x40px 圓形
│         │  背景: #F0EBE3
└─────────┘  圖標: #A67C52
```

- 尺寸：40x40px（小型）、48x48px（標準）、56x56px（大型）
- 背景：淺米色 `#F0EBE3`
- 圖標顏色：銅棕色 `#A67C52`

### 2.2 任務卡片 (TaskCard)

```
┌────────────────────────────────────────────┐
│  (○) │ 任務標題              │   +50 XP   │
│      │ 任務描述              │            │
└────────────────────────────────────────────┘
```

- 背景：白色，圓角 16px
- 左側：40px 圖標圓形
- 中間：標題（深棕 16px 600）+ 副標題（淺棕 13px）
- 右側：XP 獎勵（銅色 14px 600）或打勾圖標

### 2.3 設定列 (SettingsRow)

**帶箭頭型**
```

└────────────────────────────────────────────┘
```

```

```

**帶徽章型**
```

│  (○)  標籤文字               (2/10)   >   │
└────────────────────────────────────────────┘
```

- 高度：56-64px
- 圖標圓形 + 標籤 + 右側控件
- Toggle 顏色：啟用=銅棕色，停用=灰色
- 徽章：圓角膠囊，淺底深字

## 4. 狀態變化

### 4.1 任務狀態

| 狀態 | 右側顯示 | 樣式 |
|------|----------|------|
| 未完成 | +XX XP | 銅色文字 |
| 已完成 | ✓ 打勾 | 銅色圓形打勾圖標 |
| 鎖定 | 🔒 | 灰色，整列變淡 |

### 4.2 Toggle 開關

| 狀態 | 樣式 |
|------|------|
| 開啟 | 背景銅色，圓點在右 |
| 關閉 | 背景灰色，圓點在左 |

### 4.3 晶片/Tab 選中

| 狀態 | 樣式 |
|------|------|
| 選中 | 實心背景（銅色），白字 |
| 未選中 | 透明背景，銅色邊框，銅字 |

### 4.4 數值變化

| 類型 | 樣式 |
|------|------|
| 正向 (+) | 綠色 `#5D8A66` |
| 負向 (-) | 紅色 `#C45C5C` |
