前端同步指令
【後端更新摘要】
新增首頁公告與活動系統，包含三種類型：一般公告、快閃活動、節日限定活動。管理員可透過後台新增、編輯、刪除。快閃/節日活動到期會自動刪除。

【API 變更】
方法	路徑	用途	權限
GET	/api/announcements	取得目前有效的公告/活動	公開
GET	/api/admin/announcements	取得所有公告/活動	管理員
POST	/api/admin/announcements	新增公告/活動	管理員
PATCH	/api/admin/announcements/:id	更新公告/活動	管理員
DELETE	/api/admin/announcements/:id	刪除公告/活動	管理員
【資料結構】
interface Announcement {
  id: number;
  type: 'announcement' | 'flash_event' | 'holiday_event';
  title: string;
  content: string;
  imageUrl?: string;      // 可選，活動圖片
  linkUrl?: string;       // 可選，點擊跳轉連結
  startDate?: Date;       // 可選，開始日期
  endDate?: Date;         // 可選，結束日期（到期自動刪除）
  isActive: boolean;      // 是否啟用
  priority: number;       // 排序優先順序（數字越大越前面）
  createdAt: Date;
  updatedAt: Date;
}
【前端需要的改動】
1. 旅客首頁（Expo App）
在首頁顯示公告與活動：

// 取得有效公告
const response = await fetch('https://gacha-travel--s8869420.replit.app/api/announcements');
const data = await response.json();
// data.announcements: Announcement[]
建議 UI：

一般公告：顯示為置頂橫幅或公告列表
快閃活動：顯示為醒目卡片，可加倒數計時
節日限定活動：顯示為特殊節日主題卡片
2. 管理後台（AdminDashboard）
新增「公告管理」功能頁：

// 取得所有公告（含已停用）
const response = await fetch('https://gacha-travel--s8869420.replit.app/api/admin/announcements', {
  headers: { 'Authorization': `Bearer ${token}` }
});
// 新增公告
await fetch('https://gacha-travel--s8869420.replit.app/api/admin/announcements', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${token}`
  },
  body: JSON.stringify({
    type: 'flash_event',        // announcement | flash_event | holiday_event
    title: '限時優惠活動',
    content: '活動內容描述...',
    imageUrl: 'https://...',    // 可選
    linkUrl: 'https://...',     // 可選
    startDate: '2025-12-20',    // 可選
    endDate: '2025-12-25',      // 可選，快閃/節日活動建議填寫
    isActive: true,
    priority: 10
  })
});
// 更新公告
await fetch(`https://gacha-travel--s8869420.replit.app/api/admin/announcements/${id}`, {
  method: 'PATCH',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${token}`
  },
  body: JSON.stringify({ title: '新標題', isActive: false })
});
// 刪除公告
await fetch(`https://gacha-travel--s8869420.replit.app/api/admin/announcements/${id}`, {
  method: 'DELETE',
  headers: { 'Authorization': `Bearer ${token}` }
});
【回傳資料格式】
// GET /api/announcements
{
  announcements: Announcement[]  // 只包含 isActive=true 且在有效期間內的公告
}
// GET /api/admin/announcements
{
  announcements: Announcement[]  // 包含所有公告
}
// POST /api/admin/announcements
{
  success: true,
  announcement: Announcement
}
// PATCH /api/admin/announcements/:id
{
  success: true,
  announcement: Announcement
}
// DELETE /api/admin/announcements/:id
{
  success: true,
  message: "Announcement deleted"
}
